# GRAPHIC PORTFOLIO CMS - COMPLETE DEVELOPMENT PLAN
=======================================================

## PROJECT OVERVIEW
==================
A single-user based graphic portfolio CMS with admin panel for managing creative assets, categories, and downloadable files. Built with Laravel 12, Bootstrap 5, and modern frontend technologies.

## TECHNOLOGY STACK
==================
- **Backend**: Laravel 12 (PHP 8.2+)
- **Frontend**: Bootstrap 5, TailwindCSS 4.0, Alpine.js
- **Database**: MySQL
- **Build Tool**: Vite 7.0.7
- **Image Gallery**: Swiper.js, GLightbox
- **Admin Panel**: Filament PHP
- **File Management**: Spatie Media Library
- **Image Processing**: Intervention Image

## DATABASE DESIGN
==================

### 1. USERS TABLE (Existing)
```sql
- id (bigint, primary key)
- name (string)
- email (string, unique)
- email_verified_at (timestamp)
- password (string)
- remember_token (string)
- created_at (timestamp)
- updated_at (timestamp)
```

### 2. CATEGORIES TABLE
```sql
- id (bigint, primary key)
- name (string, unique)
- slug (string, unique)
- description (text, nullable)
- icon (string, nullable)
- color (string, default: #007bff)
- is_active (boolean, default: true)
- sort_order (integer, default: 0)
- created_at (timestamp)
- updated_at (timestamp)
```

### 3. PROJECTS TABLE
```sql
- id (bigint, primary key)
- title (string)
- slug (string, unique)
- description (text)
- short_description (string, 255 chars)
- category_id (foreign key -> categories.id)
- type (enum: 'free', 'paid')
- price (decimal 8,2, nullable)
- thumbnail_image (string, nullable)
- preview_images (json, nullable)
- download_count (integer, default: 0)
- is_featured (boolean, default: false)
- is_active (boolean, default: true)
- meta_title (string, nullable)
- meta_description (text, nullable)
- created_at (timestamp)
- updated_at (timestamp)
```

### 4. PROJECT_FILES TABLE
```sql
- id (bigint, primary key)
- project_id (foreign key -> projects.id)
- file_type (enum: 'image', 'source')
- file_name (string)
- original_name (string)
- file_path (string)
- file_size (bigint)
- mime_type (string)
- is_primary (boolean, default: false)
- created_at (timestamp)
- updated_at (timestamp)
```

### 5. SLIDERS TABLE
```sql
- id (bigint, primary key)
- title (string)
- subtitle (string, nullable)
- description (text, nullable)
- image_path (string)
- button_text (string, nullable)
- button_link (string, nullable)
- is_active (boolean, default: true)
- sort_order (integer, default: 0)
- created_at (timestamp)
- updated_at (timestamp)
```

### 6. PROFILE_SETTINGS TABLE
```sql
- id (bigint, primary key)
- name (string)
- title (string)
- bio (text)
- bio_short (string, 255 chars)
- profile_image (string, nullable)
- email (string)
- phone (string, nullable)
- address (text, nullable)
- website (string, nullable)
- linkedin_url (string, nullable)
- twitter_url (string, nullable)
- instagram_url (string, nullable)
- dribbble_url (string, nullable)
- behance_url (string, nullable)
- skills (json, nullable)
- created_at (timestamp)
- updated_at (timestamp)
```

### 7. MEDIA TABLE (Spatie Media Library)
```sql
- id (bigint, primary key)
- model_type (string)
- model_id (bigint)
- uuid (string, nullable)
- collection_name (string)
- name (string)
- file_name (string)
- mime_type (string)
- disk (string)
- conversions_disk (string)
- size (bigint)
- manipulations (json)
- custom_properties (json)
- generated_conversions (json)
- responsive_images (json)
- order_column (integer, nullable)
- created_at (timestamp)
- updated_at (timestamp)
```

## PACKAGE INSTALLATION
======================

### Backend Packages
```bash
composer require filament/filament
composer require spatie/laravel-medialibrary
composer require intervention/image
composer require laravel/breeze
```

### Frontend Packages
```bash
npm install bootstrap @popperjs/core
npm install swiper glightbox
npm install alpinejs
```
